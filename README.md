# Usage 

1. Use `Vega: Create Vega Spec` command from `View -> Command Pallette...` menu 
to Create & Save new Vega || Vega-Lite document with the corresponding Vega `json` `$schema` reference

2. Run `Vega: Preview Vega Graph` command on an open `.vg.json` || `.vl.json` Vega spec document to Preview ðŸ“ˆ

3. Save updated Vega spec `json` document to Preview updated Graph ðŸ“ˆ
